## About Me

Software Engineer with a Bachelor of Software Engineering (Honours). I'm
passionate about building reliable, maintainable and scalable solutions.

My current interests are in **backend development**, **machine learning**, and
**compiler design**. I enjoy working with Java, Go, Python, and modern web
technologies to create reliable, scalable and maintainable systems.

Currently, I'm developing a **Go CLI application** with
[Bubble Tea](https://github.com/charmbracelet/bubbletea) to identify common
vulnerabilities in programs - following my current interest in building
developer tooling.

## Featured Projects

### [CD24 Compiler]

**A custom programming language compiler with full implementation**

- Built from scratch in **Java** with a custom stack machine backend
- Implements lexical analysis, parsing, semantic analysis, and code generation
- Complete compiler toolchain for the CD24 language
- [View on GitHub](https://github.com/T-H-0-M)

<br>

### [Spiral Classification]

**Machine learning classification using SVM and Neural Networks**

- Implements and compares **Support Vector Machine** and **Neural Network**
  models
- Demonstrates training, evaluation, and visualization of ML models
- Built with **Python** and popular ML libraries

<br>

### [CD24 Syntax Highlighter]

**Syntax highlighting support for the CD24 programming language**

- **VSCode extension** providing full syntax highlighting and language support
- **Neovim autocmd** for vim users
- Built with **TypeScript** (VSCode) and **Lua** (Neovim)
